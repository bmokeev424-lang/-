# Бусидо - Клуб Восточных Единоборств

Современный сайт для клуба восточных единоборств с интеграцией Telegram бота для обработки заявок.

## Функционал

- 🥋 4 направления: Каратэ, Дзюдо, Кунг-фу, Тхэквондо
- 📅 Расписание занятий
- 💰 Тарифы и абонементы
- 🖼️ Галерея
- ⭐ Отзывы
- 📱 Telegram бот для заявок

## Настройка Telegram бота

### 1. Создание бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Придумайте название бота (например, "Бусидо Заявки")
4. Придумайте username бота (например, `busido_applications_bot`)
5. Скопируйте полученный **токен бота**

### 2. Получение Chat ID

1. Найдите [@userinfobot](https://t.me/userinfobot) в Telegram
2. Отправьте любое сообщение боту
3. Скопируйте ваш **ID** (число, например: `123456789`)

### 3. Настройка .env файла

```bash
# Скопируйте пример
cp .env.example .env

# Отредактируйте файл
nano .env
```

Заполните переменные:
```env
TELEGRAM_BOT_TOKEN=your_bot_token_here
TELEGRAM_ADMIN_CHAT_ID=your_chat_id_here
```

### 4. Запуск сервера

```bash
# Установка зависимостей
npm install

# Запуск сервера (порт 3001)
node server.js
```

Сервер запустится на `http://localhost:3001`

### 5. Запуск фронтенда

```bash
# В отдельном терминале
npm run dev
```

Фронтенд запустится на `http://localhost:5173`

## Работа с заявками

Когда пользователь отправляет заявку на сайте:

1. Данные отправляются на сервер
2. Сервер отправляет сообщение в Telegram администратору
3. В сообщении есть кнопки:
   - ✅ **Одобрить** - заявка одобрена, нужно связаться с клиентом
   - ❌ **Отклонить** - заявка отклонена
   - 📞 **Позвонить** - показывает номер телефона клиента

## Деплой

### Фронтенд (Vercel/Netlify)

```bash
npm run build
```

Загрузите папку `dist` на хостинг.

### Бэкенд (сервер)

Для production рекомендуется использовать:
- [Railway](https://railway.app)
- [Render](https://render.com)
- [Heroku](https://heroku.com)

Не забудьте установить переменные окружения на сервере!

## Технологии

- **Frontend**: React + TypeScript + Tailwind CSS + Vite
- **Backend**: Node.js + Express
- **Bot**: node-telegram-bot-api
- **Animations**: GSAP
